## 5. Tácticas accionables para stack Hispano-US

**Aplicable:**

- **AI search attribution gap** (Abr 2026 article): los modelos de attribution last-click están rotos para LLM-driven discovery. Implicación para nuestro stack: si entramos a creator/affiliate post-2026, exigir attribution beyond last-click o aceptar undercount.
- **Honey scandal (Ene 2025):** entender el riesgo de browser extensions interceptando commissions. Relevante si trabajamos con publishers o creators con audiences US grandes.
- **Linkby model (Jul 2025):** PR-to-affiliate hybrid valida la tesis de Dreamday/KMO Consulting (cubierto por Modern Retail). Convergencia PR + affiliate = tendencia confirmada.
- **Forbes Contributors fallout (Dic 2024):** confirma nuestro filtro metodológico — Forbes Contributors NO cuenta como coverage no-affiliate. Validar regla.

**NO aplicable / específica otro stack:**

- Programmatic IO content — irrelevante para affiliate puro
- Enterprise SaaS platforms (Partnerize, Impact.com) — costo y mínimo client size fuera de nuestro rango
- Publisher-side affiliate (Linkby, The Guardian) — modelo distinto al ClickBank affiliate
